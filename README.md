# ToolsFinder App

Industrial tools search and management platform built with Next.js 15.

## Features

- Search industrial tools from verified suppliers
- Personal library to save favorite tools
- User authentication (register/login)
- API routes connected to MySQL database

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Configure environment variables in `.env.local`

3. Run the database schema in `prisma/schema.sql`

4. Start development server:
   ```bash
   npm run dev
   ```

## Tech Stack

- Next.js 15 (App Router)
- MySQL with mysql2
- JWT Authentication
- StackNex Design System
